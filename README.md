# 📚 CS 학습 & 면접 대비 문제은행

1년차부터 10년차 개발자까지, 코딩 테스트와 기술 면접을 위한 **600+ CS 문제**를 제공하는 학습 플랫폼입니다.

## 🎯 프로젝트 소개

### 대상
- 신입 / 주니어 / 미들 / 시니어 개발자
- 코딩 테스트 준비생
- 기술 면접 준비생

### 특징
- **600+ 문제**: 6개 CS 분야, 총 600개 이상의 문제
- **4단계 난이도**: Level 1(기초) ~ Level 4(아키텍처)
- **실전 답변 예시**: 1분/3분 면접 답변 포함
- **꼬리 질문**: 심화 학습을 위한 추가 질문
- **검색 기능**: 키워드로 원하는 문제 바로 찾기
- **큰 글씨**: 눈이 편한 20px 기본 폰트 (조절 가능)

## 🚀 실행 방법

### 방법 1: 원클릭 실행 (추천)

1. Finder에서 프로젝트 폴더 열기
2. **`CS학습시작.command`** 파일 더블클릭
3. 자동으로 브라우저가 열리며 학습 시작!

> **참고**: 처음 실행 시 "확인되지 않은 개발자" 경고가 뜰 수 있습니다.
> - 파일 우클릭 → "열기" 선택 → "열기" 버튼 클릭

### 방법 2: 터미널 직접 실행

```bash
# 프로젝트 폴더로 이동
cd /Users/songhyeon-u/Desktop/untitled

# 서버 실행
python3 -m http.server 8000
```

그 후 브라우저에서 접속:
```
http://localhost:8000
```

### 종료 방법
- 원클릭 실행: 터미널 창 닫기
- 직접 실행: `Ctrl + C`

## 📖 사용 방법

### 1. 메인 화면
- 6개 CS 분야 카드 중 학습할 분야 클릭

### 2. 검색 기능
- 상단 검색창에 키워드 입력 (예: 트랜잭션, TCP, 프로세스)
- 2글자 이상 입력 시 실시간 검색 결과 표시
- 결과 클릭 시 해당 문제로 바로 이동

### 3. 글자 크기 조절
- 우측 하단 `A+` / `A-` 버튼으로 조절
- 기본 20px, 14px ~ 32px 범위

### 4. 스크롤
- 우측 하단 `↑` 버튼으로 맨 위로 이동

## 📝 문제 형식

각 문제는 다음 구조로 구성되어 있습니다:

```markdown
### Q1. 질문 제목

#### A. 핵심 요약
한 줄로 정리된 핵심 답변

#### 상세 설명
- 세부 내용 1
- 세부 내용 2
- 세부 내용 3

#### 실무 예시
- 실제 적용 사례

#### 면접 답변 예시
**1분**: 간단한 답변
**3분**: 상세한 답변

#### 꼬리 질문
- 심화 질문 1?
- 심화 질문 2?
```

1. **Level 1부터 시작**: 기초가 탄탄해야 심화 내용 이해 가능

3. **꼬리 질문 대비**: 면접관은 꼬리 질문을 자주 합니다
4. **실무 예시 숙지**: "실제로 사용해 본 적 있나요?"에 대비
5. **검색 활용**: 모르는 용어가 나오면 바로 검색
