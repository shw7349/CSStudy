## 🚀 실행 방법

### 방법 1: GitHub에서 바로 보기
위의 **"문제 보기"** 링크를 클릭하면 GitHub에서 바로 볼 수 있습니다.

### 방법 2: 원클릭 실행 (로컬 웹 뷰어)

1. 저장소 클론
```bash
git clone https://github.com/YOUR_USERNAME/CSStudy.git
cd CSStudy
```

2. **`CS학습시작.command`** 파일 더블클릭
3. 자동으로 브라우저가 열리며 학습 시작!

> **참고**: 처음 실행 시 "확인되지 않은 개발자" 경고가 뜰 수 있습니다.
> - 파일 우클릭 → "열기" 선택 → "열기" 버튼 클릭

### 방법 3: 터미널 직접 실행

```bash
# 프로젝트 폴더로 이동
cd CSStudy

# 서버 실행
python3 -m http.server 8000
```

그 후 브라우저에서 접속: http://localhost:8000

## 📖 사용 방법

### 웹 뷰어 기능
- **검색**: 상단 검색창에 키워드 입력 (예: 트랜잭션, TCP, 프로세스)
- **글자 크기**: 우측 하단 `A+` / `A-` 버튼
- **맨 위로**: 우측 하단 `↑` 버튼

## 📝 문제 형식

각 문제는 다음 구조로 구성되어 있습니다:

```
### Q1. 질문 제목

#### A. 핵심 요약
한 줄로 정리된 핵심 답변

#### 상세 설명
- 세부 내용 1
- 세부 내용 2

#### 실무 예시
- 실제 적용 사례

#### 면접 답변 예시
**1분**: 간단한 답변
**3분**: 상세한 답변

#### 꼬리 질문
- 심화 질문 1?
- 심화 질문 2?
```

## 💡 학습 팁

1. **Level 1부터 시작**: 기초가 탄탄해야 심화 내용 이해 가능
2. **1분 답변 먼저**: 핵심을 간결하게 말하는 연습
3. **꼬리 질문 대비**: 면접관은 꼬리 질문을 자주 합니다
4. **실무 예시 숙지**: "실제로 사용해 본 적 있나요?"에 대비
5. **검색 활용**: 모르는 용어가 나오면 바로 검색
