## 📝 문제 구성 포맷

모든 문제는 아래 구조를 따릅니다:

```markdown
## Q. 질문

### A. 핵심 요약

### 상세 설명

### 실무 예시

### 면접 답변 예시 (1분 / 3분)

### 꼬리 질문 & 심화 질문
```

## 🚀 학습 가이드

### 신입 개발자 (1~2년차)
1. 각 분야 Level 1 문제부터 시작
2. 핵심 요약과 상세 설명에 집중
3. 면접 답변 예시 1분 버전 암기

### 주니어 개발자 (3~4년차)
1. Level 1~2 문제 완주
2. 실무 예시와 연결하여 학습
3. 꼬리 질문 대비

### 미들 개발자 (5~7년차)
1. Level 2~3 문제 집중
2. 심화 구조 이해 및 트레이드오프 파악
3. 면접 답변 3분 버전 준비

### 시니어 개발자 (8~10년차)
1. Level 3~4 문제 마스터
2. 아키텍처 설계 및 트러블슈팅 관점
3. 후배 개발자 멘토링 자료로 활용
